git clone 원격 저장소의 모든 내용을 로컬 git re저장소로 복사

git remote 로컬 저장소를 원격 저장소와 연결

git push 로컬 저장소의 내용을 보내거나 로컬 저장소의 변경사항을 원격 저장소로 보냄

git pull    git remote명령을 통해 서로 연결저장소의 최신내용을 로컬 저장소로 가져오면서
병합하고 git push와 반대 성격의 명령어


git fetch 로컬 저장소와 원격 저장소의 변경 사항이 다를 때 이를 비교 대조하고 git merge명령
어와 함께 최신 데이터를 반영하거나 충돌 문제등을 해결
